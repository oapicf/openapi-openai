OpenAI API

Version: 2.0.0

APIs for sampling from and fine-tuning language models

Schemes:

Summary

Path Operation Description
/audio/transcriptions POST

Transcribes audio into the input language.
/audio/translations POST

Translates audio into English.
/chat/completions POST

Creates a model response for the given chat conversation.
/completions POST

Creates a completion for the provided prompt and parameters.
/edits POST

Creates a new edit for the provided input, instruction, and parameters.
/embeddings POST

Creates an embedding vector representing the input text.
/files GET

Returns a list of files that belong to the user's organization.
POST

Upload a file that contains document(s) to be used across various endpoints/features. Currently, the size of all the files uploaded by one organization can be up to 1 GB. Please contact us if you need to increase the storage limit.
/files/{file_id} DELETE

Delete a file.
GET

Returns information about a specific file.
/files/{file_id}/content GET

Returns the contents of the specified file
/fine-tunes GET

List your organization's fine-tuning jobs
POST

Creates a job that fine-tunes a specified model from a given dataset.

Response includes details of the enqueued job including job status and the name of the fine-tuned models once complete.

Learn more about Fine-tuning
/fine-tunes/{fine_tune_id} GET

Gets info about the fine-tune job.

Learn more about Fine-tuning
/fine-tunes/{fine_tune_id}/cancel POST

Immediately cancel a fine-tune job.
/fine-tunes/{fine_tune_id}/events GET

Get fine-grained status updates for a fine-tune job.
/images/edits POST

Creates an edited or extended image given an original image and a prompt.
/images/generations POST

Creates an image given a prompt.
/images/variations POST

Creates a variation of a given image.
/models GET

Lists the currently available models, and provides basic information about each one such as the owner and availability.
/models/{model} DELETE

Delete a fine-tuned model. You must have the Owner role in your organization.
GET

Retrieves a model instance, providing basic information about the model such as the owner and permissioning.
/moderations POST

Classifies if text violates OpenAI's Content Policy

Whether to stream events for the fine-tune job. If set to true, events will be sent as data-only server-sent events as they become available. The stream will terminate with a data: [DONE] message when the job is finished (succeeded, cancelled, or failed).

If set to false, only events generated so far will be returned.
