RunObject

  • id : String
    The identifier, which can be referenced in API endpoints.
  • object : String
    The object type, which is always `thread.run`.
    Enum:
    thread.run
  • createdUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run was created.
  • threadUnderscoreid : String
    The ID of the [thread](/docs/api-reference/threads) that was executed on as a part of this run.
  • assistantUnderscoreid : String
    The ID of the [assistant](/docs/api-reference/assistants) used for execution of this run.
  • status : String
    The status of the run, which can be either `queued`, `in_progress`, `requires_action`, `cancelling`, `cancelled`, `failed`, `completed`, or `expired`.
    Enum:
    queued
    in_progress
    requires_action
    cancelling
    cancelled
    failed
    completed
    expired
  • requiredUnderscoreaction : RunObject_required_action
  • lastUnderscoreerror : RunObject_last_error
  • expiresUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run will expire.
  • startedUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run was started.
  • cancelledUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run was cancelled.
  • failedUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run failed.
  • completedUnderscoreat : Integer
    The Unix timestamp (in seconds) for when the run was completed.
  • incompleteUnderscoredetails : RunObject_incomplete_details
  • model : String
    The model that the [assistant](/docs/api-reference/assistants) used for this run.
  • instructions : String
    The instructions that the [assistant](/docs/api-reference/assistants) used for this run.
  • tools : List
    The list of tools that the [assistant](/docs/api-reference/assistants) used for this run.
  • fileUnderscoreids : List
    The list of [File](/docs/api-reference/files) IDs the [assistant](/docs/api-reference/assistants) used for this run.
  • metadata : Object
    Set of 16 key-value pairs that can be attached to an object. This can be useful for storing additional information about the object in a structured format. Keys can be a maximum of 64 characters long and values can be a maxium of 512 characters long.
  • usage : RunCompletionUsage
  • temperature : BigDecimal
    The sampling temperature used for this run. If not set, defaults to 1.
  • maxUnderscorepromptUnderscoretokens : Integer
    The maximum number of prompt tokens specified to have been used over the course of the run.
  • maxUnderscorecompletionUnderscoretokens : Integer
    The maximum number of completion tokens specified to have been used over the course of the run.
  • truncationUnderscorestrategy : TruncationObject
  • toolUnderscorechoice : AssistantsApiToolChoiceOption
  • responseUnderscoreformat : AssistantsApiResponseFormatOption
