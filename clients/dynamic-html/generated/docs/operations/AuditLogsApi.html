This is the API

listAuditLogs

URL

https://api.openai.com/v1/organization/audit_logs

HTTP Method

GET

Response Type

ListAuditLogsResponse

Parameters

  • Query: effective_at ListAuditLogsEffectiveAtParameter

    Return only events whose `effective_at` (Unix seconds) is in this range.

  • Query: project_ids[] List(String)

    Return only events for these projects.

  • Query: event_types[] List(AuditLogEventType)

    Return only events with a `type` in one of these values. For example, `project.created`. For all options, see the documentation for the [audit log object](/docs/api-reference/audit-logs/object).

  • Query: actor_ids[] List(String)

    Return only events performed by these actors. Can be a user ID, a service account ID, or an api key tracking ID.

  • Query: actor_emails[] List(String)

    Return only events performed by users with these emails.

  • Query: resource_ids[] List(String)

    Return only events performed on these targets. For example, a project ID updated.

  • Query: limit Integer

    A limit on the number of objects to be returned. Limit can range between 1 and 100, and the default is 20.

  • Query: after String

    A cursor for use in pagination. `after` is an object ID that defines your place in the list. For instance, if you make a list request and receive 100 objects, ending with obj_foo, your subsequent call can include after=obj_foo in order to fetch the next page of the list.

  • Query: before String

    A cursor for use in pagination. `before` is an object ID that defines your place in the list. For instance, if you make a list request and receive 100 objects, starting with obj_foo, your subsequent call can include before=obj_foo in order to fetch the previous page of the list.
