createImageEdit

URL

https://api.openai.com/v1/images/edits

HTTP Method

POST

Response Type

ImagesResponse

Parameters

  • Form: image File(file)

    The image to edit. Must be a valid PNG file, less than 4MB, and square. If mask is not provided, image must have transparency, which will be used as the mask.

  • Form: prompt String(string)

    A text description of the desired image(s). The maximum length is 1000 characters.

  • Form: mask File(file)

    An additional image whose fully transparent areas (e.g. where alpha is zero) indicate where `image` should be edited. Must be a valid PNG file, less than 4MB, and have the same dimensions as `image`.

  • Form: model CreateImageEditRequest_model(CreateImageEditRequest_model)

  • Form: n Integer(integer)

    The number of images to generate. Must be between 1 and 10.

  • Form: size String(string)

    The size of the generated images. Must be one of `256x256`, `512x512`, or `1024x1024`.

  • Form: response_format String(string)

    The format in which the generated images are returned. Must be one of `url` or `b64_json`. URLs are only valid for 60 minutes after the image has been generated.

  • Form: user String(string)

    A unique identifier representing your end-user, which can help OpenAI to monitor and detect abuse. [Learn more](/docs/guides/safety-best-practices/end-user-ids).

createImageVariation

URL

https://api.openai.com/v1/images/variations

HTTP Method

POST

Response Type

ImagesResponse

Parameters

  • Form: image File(file)

    The image to use as the basis for the variation(s). Must be a valid PNG file, less than 4MB, and square.

  • Form: model CreateImageEditRequest_model(CreateImageEditRequest_model)

  • Form: n Integer(integer)

    The number of images to generate. Must be between 1 and 10. For `dall-e-3`, only `n=1` is supported.

  • Form: response_format String(string)

    The format in which the generated images are returned. Must be one of `url` or `b64_json`. URLs are only valid for 60 minutes after the image has been generated.

  • Form: size String(string)

    The size of the generated images. Must be one of `256x256`, `512x512`, or `1024x1024`.

  • Form: user String(string)

    A unique identifier representing your end-user, which can help OpenAI to monitor and detect abuse. [Learn more](/docs/guides/safety-best-practices/end-user-ids).
